# Okanvil-IDs

Search spells & items **by name → get the ID** for WotLK **3.3.5a**. Great for building WeakAuras (which need spell/item IDs). Type a name, get matching results with their IDs and icons.

Runs **standalone**, or docks into the [Okanvil](https://github.com/MrNog/Okanvil) host when installed.

## Install

1. Download the **.zip**, extract, and drop the **`Okanvil-IDs`** folder into `World of Warcraft\Interface\AddOns\` (remove any `-main` suffix).
2. Restart WoW or `/reload`.
3. Commands: **`/cid`** or **`/idfind`**.

---

Interface: **3.3.5a (30300)**.
