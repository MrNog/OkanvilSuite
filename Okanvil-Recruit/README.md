# Okanvil-Recruit

A generic **guild-recruitment advertiser** for WotLK **3.3.5a**: auto-advertise, auto-reply (AFK mode), and auto-invite, with keyword + known-contact filters. Tabbed UI (Text / Settings / Filters / Whispers / Summary) and a minimap button.

The guild name is **configurable** — use the `{guild}` token in any message and it's replaced at send time. Nothing is pre-filled; you write all the text.

Runs **standalone**, or docks into the [Okanvil](https://github.com/MrNog/Okanvil) host when installed.

## Install

1. Download the **.zip**, extract, and drop the **`Okanvil-Recruit`** folder into `World of Warcraft\Interface\AddOns\` (remove any `-main` suffix).
2. Restart WoW or `/reload`.
3. Commands: **`/recruit`** or **`/rec`**.

## Quick start

1. `/recruit` → **Text** tab → write your advert (use `{guild}` where you want the guild name).
2. **Settings** → set your guild name + channels/interval.
3. **Filters** → keywords / blacklist as needed.
