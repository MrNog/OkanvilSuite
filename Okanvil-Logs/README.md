# Okanvil-Logs

Combat-log control + **REC** timer + session tracker for WotLK **3.3.5a**.

Turns combat logging on/off cleanly, shows a recording indicator/timer, and tracks your raid session. Runs **standalone**, or docks into the [Okanvil](https://github.com/MrNog/Okanvil) host when it's installed (shared config window + theme).

## Install

1. Download the **.zip**, extract, and drop the **`Okanvil-Logs`** folder into `World of Warcraft\Interface\AddOns\` (remove any `-main` suffix).
2. Restart WoW or `/reload`.
3. Command: **`/cflog`**.

Optionally install the [Okanvil](https://github.com/MrNog/Okanvil) host to manage it (and other plugins) from one window.

---

Interface: **3.3.5a (30300)**.
